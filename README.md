# Video Game Center
A comprehensive game info website serves as a rich and detailed resource for video game enthusiasts. The
platform is dedicated to providing up-to-date information on multiple platforms.

## Getting Started

### Prerequisites
- Node.js

### Installation
1. cd game-hub
2. npm install

### Usage
- npm run dev
